# Zapaste UI

Zapaste UI is a web front-end interface for [Zapaste][zapaste]. The overall system is based on Vite + React, and the UI style is based on [Material Design 3][material3].

## Contents

- [Zapaste UI](#zapaste-ui)
  - [Contents](#contents)
  - [Features](#features)
  - [Pictures](#pictures)

## Features

- Public pastebin list
- Pastebin viewing
- Pastebin editing
- File upload and download
- Global permission verification support
- Mobile device compatibility
- Built-in web server available in Zapaste
