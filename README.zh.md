# Zapaste UI

Zapaste UI 是为 [Zapaste][zapaste] 所制作的一个 Web 前端界面. 整体基于 Vite + React, UI 风格基于 [Material Design 3][material3].

## 目录

- [Zapaste UI](#zapaste-ui)
  - [目录](#目录)
  - [支持功能](#支持功能)
  - [界面展示](#界面展示)

## 支持功能

- 公共剪切板列表
- 剪切板阅览
- 剪切板编辑
- 文件上传与下载
- 全局权限校验支持
- 移动端适配
- Zapaste 内嵌Web服务器可用
